How to successfully build this application:
1) File -> Import Module... -> <android-sdk>\extras\google\google_play_services\libproject\google-play-services_lib
2) File -> Import Module... -> <android-sdk>\extras\android\support\v7\appcompat
3) Ctrl + Shift + Alt + S -> Modules -> <Project Name> -> Dependencies -> Add -> Module Dependency -> google-play-services_lib
4) Add -> Jars or directories -> <android-sdk>\extras\google\google_play_services\libproject\google-play-services_lib\libs
5) Add -> Jars or directories -> <android-sdk>\extras\android\support\v4
6) Add -> Module Dependency -> appcompat
